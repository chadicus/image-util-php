    resizeMulti(\Imagick $source, array $boxSizes, array $options = array()) : array
    Static

    Parameters

    $source

    \Imagick

    source image to resize. Will not modify

    $boxSizes

    array

    resulting bounding boxes. Each value should be an array with width and height, both integers

    $options

    array

    options

    string color (default white) background color. Any supported from
    http://www.imagemagick.org/script/color.php#color_names
bool upsize (default false) true to upsize the original image or false to upsize just the bounding box
int maxWidth (default 10000) max width allowed for $boxWidth
int maxHeight (default 10000) max height allowed for $boxHeight

    Exceptions

    \InvalidArgumentException if $options["color"] was not a string
    \InvalidArgumentException if $options["upsize"] was not a bool
    \InvalidArgumentException if $options["maxWidth"] was not an int
    \InvalidArgumentException if $options["maxHeight"] was not an int
    \InvalidArgumentException if a width in a $boxSizes value was not an int
    \InvalidArgumentException if a height in a $boxSizes value was not an int
    \InvalidArgumentException if a $boxSizes width was not between 0 and $options["maxWidth"]
    \InvalidArgumentException if a $boxSizes height was not between 0 and $options["maxHeight"]
    \Exception

    Returns

    arrayarray of \Imagick objects resized. Keys maintained from $boxSizes
